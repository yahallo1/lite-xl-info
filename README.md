# Lite-XL Info Reader
Plugin that allows you to read info pages inside [Lite-XL](https://github.com/lite-xl/lite-xl)

## Dependencies
You need to have the `info` binary installed on your system.

## Commands
- `core:open-info` Open an info manual

- `info:up-node` Go to parent node

- `info:next-node` Go to next node

- `info:previous-node` Go to previous node

- `info:go-back-node` Go to last opened node
